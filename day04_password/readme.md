# 🔐 密码管理器

这是我的第4个Python练习项目，一个简单的命令行密码管理器。

## 📋 功能特点

- ✅ **添加密码** - 记录网站名称、网址、用户名、密码、备注、标签
- ✅ **查看所有密码** - 表格形式显示，密码自动隐藏为星号
- ✅ **查看单个密码** - 可查看完整信息，支持选择是否显示密码原文
- ✅ **修改密码** - 支持修改任意字段（网站/网址/用户名/密码/备注/标签）
- ✅ **删除密码** - 按ID删除不需要的密码
- ✅ **数据持久化** - 自动保存到 passwords.json 文件

## 🛠️ 技术要点

| 技术 | 用途 |
|------|------|
| Python 基础 | 核心逻辑实现 |
| 字典 + 列表 | 数据结构存储 |
| JSON 模块 | 文件读写 |
| datetime | 记录创建时间 |
| time | 生成唯一ID |
