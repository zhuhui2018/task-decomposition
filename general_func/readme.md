
# 拆解思维
以备份日志为例

#1，这个功能的作用是什么
#备份文件，防止重要文件丢失

#2，怎么实现，也就是说需要什么功能

a）需要有源文件，目标存储地址，考虑到内存，文件个数不能太多，
所以要有一个文件存储个数，这些都放在初始化中
b) 要有备份功能（核心），把源文件备份一个存放在目标路径中，
那就会涉及到源文件和目标文件路径处理等问题
c）要有文件列表和删除功能，因为要知道有多少个备份文件，
如果文件超过10个要进行删除，那这就会涉及到排序功能，按照
最新的文件在前面排序，把10个后面的文件删除掉
d）需要一个恢复功能，比如文件修改错了有问题，得指定恢复哪个文件

## 2024-03-09 备份工具复盘

### 拆解过程
1. 初始化需要：源文件、备份目录、保留数量
2. 起名字需要：源文件名、时间戳
3. 执行备份：复制文件
4. 清理：找备份、删旧的
5. 列表：显示备份
6. 恢复：复制回来

- 写完要检查方法调用链
